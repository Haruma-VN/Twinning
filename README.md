# TwinKleS ToolKit

Powerful tool for PvZ-2 Modder .

by TwinKleS-China .

## 开源说明

本项目使用 GPL 协议，可以随意使用与修改，但使用本项目的项目也须开源并使用 GPL 协议。

## 模块组成

* Core
	
	核心功能的实现，以动态库形式分发。

* Shell
	
	面向用户的外壳程序，提供基于终端的命令行界面。

* Script
	
	与 Core 及 Shell 配合使用的脚本。

* Windows Explorer Extension
	
	适用于 Windows 的资源管理器拓展，将 ToolKit 集成至系统资源管理器的右键菜单中。

## 平台支持

* 操作系统
	
	* Windows | Linux | MacOS
	
	* Android | iOS

* 处理器架构
	
	* x86_32 | x86_64
	
	* arm_32 | arm_64

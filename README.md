# TwinKleS.ToolKit

by TwinKleS-China

## 模块

* Core 核心
	
	核心功能的实现。

* Shell 外壳
	
	命令行 UI。
	
* Script 脚本
	
	与 Core 及 Shell 配合使用的脚本。
	
* WindowsExplorerExtension Windows资源管理器拓展
	
	适用于 Windows 的资源管理器拓展，将 ToolKit 集成至系统资源管理器的右键菜单中。
